<h1>Clube do livro virtual</h1>

+ Tema do projeto: O projeto tem como tema um clube do livro virtual, ou seja, é inspirado em comunidades como Discord e aplicativos de leituras como Bookshelf 
+ Área do projeto: Linguagens e técnico

<h2>Projeto</h2>

+ Este projeto visa incentivar a leitura e a comunicação entre o público jovem (público alvo), através de clubes e livros. Afinal hoje em dia estudos mostram que 66,7% dos jovens não leem textos com mais de 10 páginas, ou seja, devemos incentivar a leitura de maneira que seja divertida, inclusiva e não obrigatória.
+ Clubes: Os clubes são públicos e privados, ou seja, vão existir grupos abertos a todos, porém existem os privados, ou seja, o usuário cria o clube e ele decide quem pode entrar nele;
+ Livros: No caso dos livros, o site conta com ideias e opções de leitura separados por gênero. Você vai conseguir colocar a quantidade lida e fazer resenhas sobre, além de comentar sobre eles nos clubes;
+ Metas e desafios: Existe uma aba somente para metas e desafios de leitura, como ler um livro em 30 dias, ou 50 páginas em 10 minutos (contando com timer). 

<h2>Objetivos:</h2>

+ O desenvolvimento do projeto busca o uso de linguagens de programação para criar um site acessível e também interativo, buscando o interesse do público alvo;
+ O projeto busca promover a cultura letrada e também a comunicação, afinal a falta da mesma pode levar a sérios riscos para a saúde mental;
+ Com esse projeto gostaria de aprimorar meus conhecimentos em várias linguagens, como HTML, CSS, JavaScript e o banco de dados MySQL;

<h2>Justificativa:</h2><p>Este projeto foi desenvolvido pensando em números de livros que os jovens leem por ano, mas principalmente pensando em saúde mental. Hoje em dia a falta de amigos e ciclos sociais são os principais motivos que desenvolvem mais ainda a depressão e ansiedade em jovens. Com isso o projeto visa juntar pessoas com intereses parecidos e com isso elas não conversarem somente sobre os livros, mas também sobre elas mesmas</p>

<h2>Ferramentas utilizadas:</h2>

+ Linguagens: HTML, CSS, JavaScript
+ Banco de dados: MySQL
+ Ambiente de desenvolvimento: VSCode
